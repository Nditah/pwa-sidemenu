This is an enhanced version of the standard Ionic SideMenu template, configured for PWA using Workbox.

Workbox is a set of libraries and Node modules that make it easy to cache assets and take full advantage of features used to build Progressive Web Apps.

Want to add Facebook/Google login, connect to Firestore, get external data using HttpClient, add Camera, Geolocation and many more? Click here to get Ionic PWA Template Resources Firebase Hosting Workbox Getting Started

  • Make sure you have an up-to-date version of Node.js installed on your system. If you don't have Node.js installed, you can install it from

    https://nodejs.org/en/download/

  • Unzip the Template file to your project directory

  • Open a terminal window (Mac) or a command window (Windows), and navigate to the project directory.

  • Install the server dependencies:

    npm install

  • Test it in your browser.

    ionic serve

  • If you follow all the Steps from 1 - 4, your apps will be displayed in your browser.

 Deploy to Firebase Hosting

  • Open a terminal window (Mac) or a command window (Windows) and go to your project directory

  • Install the Firebase CLI.

    npm install -g firebase-tools

  • Initialize your site

    firebase init

  • On subsequent build and deploy, you do not need to do the above 2 steps.

  • Build your project

    npm run www

  • Deploy your site. Upon completion, go to the URL provided.

    firebase deploy

  • CONGRATULATION. You PWA is now LIVE.
